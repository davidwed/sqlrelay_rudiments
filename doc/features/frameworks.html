Intuitive Frameworks

Rudiments provides intuitive frameworks for developing applications and systems.

Applications

Process the command line using the commandline class. Access environment variables using the environment class. Control the process environment using the process class.

Use the process and thread classes to spawn and manage child processes and threads. Synchronize threads with the threadmutex class. Synchronize and communicate with other processes using the semaphoreset, sharedmemory, and signal classes.

Clients

Build client programs using the inetsocketclient and unixsocketclient classes.

Servers

Build server software using the inetsocketserver, unixsocketserver, and listener classes.

XML

Process XML using SAX or DOM paradigms with the xmlsax, xmldom, and xmldomnode classes.

Logging

Log to standard out, standard error, files or syslog using the logger class and log destination framework.

Compilers

Use an XML grammar with the codetree class to parse code into a DOM tree, and develop modules to trans-compile it with the compiler framework.
