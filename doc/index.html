Rudiments

Rudiments is a low-level C++ class library for developing OS-agnostic applications.

With Rudiments, you can develop applications using intuitive frameworks for daemons, clients and servers, lightweight data structures, and comprehensive utility classes.



Advantages

Intuitive Frameworks - from clients and servers to a compiler framework
Lightweight Data Structures - string and byte buffers, linked lists, and dictionaries
Comprehensive Utility Classes - threads and processes, regular expressions, dates and times...

Lightweight Implementation - stays close to native code
"Better C" - shallow heirachies, minimal templates
Compatibility - glibc, uClibc, BSD C, Win32...
