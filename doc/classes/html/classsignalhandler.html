static bool signalhandler::isSignalHandlerIntUsed ( )
static

The function that you pass into setHandler() must have a void (*)(int32_t) signature. However, the native signal handling frameworks on some systems do not allow the signal number to be passed to the handler in the int32_t parameter. As a result, on those platforms, inside the signal handler, the paramer must be ignored.

This method will return true if the platform's native signal handling framework passes anything into the int32_t parameter and false otherwise.
